# Tech Stack

## 1. Frontend
- **Framework:** ReactJS (cấu trúc với Create React App hoặc Vite).
- **Ngôn ngữ:** JavaScript / JSX.
- **Routing:** React Router DOM.
- **State Management:** Context API (AuthContext, ThemeContext, ...).
- **Styling:** CSS thuần hoặc TailwindCSS / CSS Modules (tùy chỉnh).
- **HTTP Client:** Axios / Fetch API.
- **Build Tool:** Vite / Webpack.

## 2. Backend
- **Framework:** Spring Boot (Java).
- **Bảo mật:** Spring Security + JWT (JSON Web Token).
- **ORM:** Spring Data JPA / Hibernate.
- **Build Tool:** Maven.

## 3. Database
- **DBMS:** MySQL hoặc PostgreSQL.

## 4. DevOps & Deployment
- Docker (Có sẵn Dockerfile, compose.yml).
- Nginx (Reverse Proxy cho Frontend).
